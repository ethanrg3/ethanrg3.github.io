Office Euchre | Class Project

Python

The Office Euchre showcases strong Object-Oriented Programming (OOP) fundamentals through its thoughtful design and implementation. The game features four core classes: Game, Deck, Card, and Player. Encapsulation is demonstrated by organizing attributes (e.g., Player’s hand, Deck’s play_cards) and related methods, ensuring modularity. Polymorphism is employed in Card with overridden comparison operators for card ranking. Inheritance-free, it emphasizes interaction between classes for game flow.
